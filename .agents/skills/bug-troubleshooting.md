# Bug Troubleshooting Skill

This skill helps the AI systematically troubleshoot and fix bugs.

## Workflow

1.  **Reproduce/Understand:** Analyze the error message, stack trace, and user description to understand how the bug manifests.
2.  **Isolate:** Pinpoint the exact file, function, or line of code causing the issue. Use search or read tools to inspect the relevant codebase sections.
3.  **Hypothesize:** Formulate a hypothesis on why the bug is occurring.
4.  **Test Hypothesis:** If applicable, propose a small test or debug log to verify the hypothesis.
5.  **Propose Fix:** Provide the exact code changes needed to resolve the bug, ensuring no regressions are introduced. Explain *why* the fix works.

# Document Codebase into Confluence Skill

This skill guides the AI to generate comprehensive technical documentation for the codebase and publish it to Confluence using the Atlassian MCP Server.

## Workflow

1.  **Analyze Scope:** Understand which part of the codebase needs documentation (e.g., a specific module, API, or the overall architecture). Read the relevant files.
2.  **Draft Content:** Create a structured document in markdown:
    *   **Overview:** High-level description of the module.
    *   **Architecture/Flow:** How the components interact.
    *   **API/Interfaces:** Key functions, classes, and data models.
    *   **Dependencies:** External libraries or services used.
3.  **Publish to Confluence:** Use the Atlassian/Confluence MCP Server tools to create a new page or update an existing page with the drafted content.
4.  **Confirm:** Return the link or confirmation of the published Confluence page to the user.

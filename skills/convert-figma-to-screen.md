# Convert Figma to Screen Skill

This skill guides the AI to translate a Figma design into frontend code (React, HTML/CSS, etc.).

## Workflow

1.  **Analyze Design:** Use the Figma MCP Server or user-provided design descriptions/screenshots to understand the layout, colors, typography, and components.
2.  **Structure Components:** Break down the design into logical, reusable frontend components.
3.  **Generate Code:**
    *   Write the markup (HTML/JSX) matching the design structure.
    *   Apply styles (Tailwind, CSS Modules, Styled Components) accurately reflecting the Figma specifications.
4.  **Add Interactivity:** If the design implies state (e.g., dropdowns, modals, hover states), implement the necessary frontend logic.
5.  **Review:** Ensure the generated code is responsive and accessible.

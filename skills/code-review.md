# Code Review Skill

This skill guides the AI to perform a comprehensive code review.

## Workflow

1.  **Analyze Context:** Read the changed files (e.g., via `git diff` or provided code snippets).
2.  **Identify Issues:** Check for bugs, security vulnerabilities, performance bottlenecks, and adherence to clean code principles (SOLID, DRY).
3.  **Suggest Improvements:** Provide actionable feedback with code examples showing the improved version.
4.  **Highlight Positives:** Mention well-written parts of the code to provide balanced feedback.
5.  **Format Output:** Present the review clearly using markdown, categorizing feedback by severity (Critical, Minor, Nitpick).

# Workshop IA - Alpha Software

This repository contains the AI skills and prompts created for the Alpha Software AI workshop.

## Directory Structure

*   `skills/`: The master directory containing all the skill definitions.
*   `.cursor/skills/`: Symlinks for Cursor IDE.
*   `.claude/skills/`: Symlinks for Claude Code.
*   `.github/copilot-instructions/`: Symlinks for GitHub Copilot.

## Skills Available

*   Code Review
*   Bug Troubleshooting
*   Create Jira Ticket for Bug
*   Convert Figma to Screen
*   Document Codebase into Confluence
*   Read Jira Ticket and Implement
*   Convert Slack Message to Jira Ticket
